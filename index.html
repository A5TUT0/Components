Bibliotecas de Componentes UI

CSS Libraries
  • Bootstrap: Popular framework de componentes basado en CSS, con una gran comunidad y ecosistema de plugins.
  • Bulma: Framework CSS ligero, moderno, y muy personalizable, basado en Flexbox.
  • Foundation: Un framework avanzado de CSS, orientado a proyectos empresariales.
  • Semantic UI: Librería con un enfoque en la semántica y la legibilidad del código.
  • Tailwind CSS: Framework de utilidad para diseñar directamente en el HTML con clases predefinidas.
Bibliotecas basadas en Tailwind CSS
  • DaisyUI: Componentes estilizados basados en Tailwind para construir interfaces rápidamente.
  • Flowbite: Biblioteca de componentes UI diseñada sobre Tailwind CSS con soporte para React, Vue y otros.
  • Headless UI: Componentes sin estilo integrados a la perfección con Tailwind CSS, compatibles con React y Vue.
  • Meraki UI: Conjunto de componentes gratuitos basados en Tailwind CSS para interfaces modernas.
Bibliotecas UI solo para React
  • Material UI: Implementación de componentes de Material Design para React.
  • Chakra UI: Componentes accesibles, modernos y flexibles para React.
  • Ant Design: Sistema de diseño de componentes para aplicaciones empresariales de React.
  • Blueprint: Componentes de interfaz de usuario diseñados para aplicaciones complejas y empresariales en React.
  • Evergreen: Librería de componentes que prioriza la personalización y la estética limpia.
  • Rebass: Conjunto de componentes basados en Styled System para React.
Bibliotecas UI solo para Angular
  • Angular Material: Componentes de Material Design implementados específicamente para Angular.
  • NG-ZORRO: Componentes UI basados en Ant Design pero optimizados para Angular.
  • PrimeNG: Colección rica en componentes UI para Angular con soporte avanzado de tablas, formularios y más.
  • Nebular: Conjunto de componentes basados en diseños modernos y con soporte para temas oscuros.
Bibliotecas UI solo para Vue
  • Vuetify: Implementación completa de Material Design para Vue.
  • Quasar Framework: Biblioteca de componentes y herramientas para crear aplicaciones Vue responsive y con un gran rendimiento.
  • Element UI: Componentes UI con un diseño minimalista y adaptado para la web moderna.
  • Buefy: Componentes basados en Bulma CSS para Vue.
  • BootstrapVue: Componentes basados en Bootstrap adaptados para Vue.js.
Bibliotecas UI para aplicaciones móviles (React Native)
  • React Native Paper: Implementación de Material Design para aplicaciones React Native.
  • NativeBase: Componentes accesibles y listos para usar en React Native.
  • React Native Elements: Biblioteca de componentes altamente personalizables y simplificados para aplicaciones móviles.
Bibliotecas basadas en Svelte
  • Svelte Material UI: Componentes UI basados en Material Design para Svelte.
  • SvelteStrap: Implementación de Bootstrap para Svelte.
  • Carbon Components Svelte: Componentes Svelte basados en el sistema de diseño de IBM Carbon.
Librerías agnósticas y multi-framework
  • PrimeReact / PrimeNG / PrimeVue: Componentes avanzados disponibles para React, Angular y Vue.
  • Carbon Design System: Sistema de diseño de IBM con implementaciones para React, Angular, Vue y otros.
  • Clarity Design System: Sistema de diseño de VMware con implementaciones para Angular y soporte de web components.
  • Onsen UI: Biblioteca para el desarrollo de aplicaciones móviles, compatible con Vue, React, Angular y Vanilla JS.
Frameworks y librerías para aplicaciones de diseño de interacción
  • Framer Motion: Biblioteca para animaciones en React.
  • Lottie: Animaciones ligeras exportadas desde After Effects y compatibles con la mayoría de los frameworks.
Otras
  • Tachyons: Biblioteca CSS que facilita la creación de interfaces mediante clases funcionales.
  • UIKit: Framework CSS modular y ligero para construir interfaces web rápidas y funcionales.
  • Materialize: Componentes basados en Material Design para aplicaciones web.
